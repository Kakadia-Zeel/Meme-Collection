# Meme Collection

A meme collection is a website where one user can submit meme url and caption.

## Description

This project was a part of  Crio's Externship Program of Stage 2B.



### Dependencies

* Front-end in React : Axios, React Bootstrap , Formik and Yup
* Backend : Express
* Database : SQLITE
